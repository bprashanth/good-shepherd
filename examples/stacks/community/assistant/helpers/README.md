# Custom Helper Scripts

This directory contains agent-created helper scripts for common nursery operations.

## Purpose

The agent can autonomously create bash scripts here when it discovers repetitive patterns or common queries.

## Usage

Helpers are sourced and called like:
```bash
source /home/node/.openclaw/workspace/frappe_helpers.sh
./helpers/your_helper.sh [args]
```
