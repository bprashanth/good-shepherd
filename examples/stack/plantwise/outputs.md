# Plantwise Outputs

This document describes the artifacts emitted by `examples/stack/plantwise/`.

## Stage 0 Contract

The final implementation of this stage is expected to emit:

- selected plot center coordinate artifacts
- curated species list artifacts
- nursery-to-species ranking artifacts

## Expected Consumers

- primarily the Plantwise UI itself
- optionally external review or downstream static integrations

## Verification Gate

Before this stage is treated as stable, it must be manually verified in the UI for:

- selected plot click behavior
- nursery display
- species ranking display

This file will be updated after Stage 4 with exact filenames and contracts.
