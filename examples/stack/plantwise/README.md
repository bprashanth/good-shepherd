# Plantwise

This is a practitioner planning interface composed from existing map and static frontend patterns already present in `examples/`.

It will:

- reuse the AOI and selected plot from site selection
- show the selected plot overlay without the earlier raster layers
- respond to plot clicks with a Plantwise-style recommendation panel
- show ranked species grouped under two nurseries

The initial species list will be prechosen from `examples/stack/indicators/input/` and stored locally under `input/`.
